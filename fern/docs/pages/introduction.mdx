# casestudy.dev

The case study API for sales teams, AI agents, and anyone who needs real-world proof of what works.

We crawl, structure, and index case studies from hundreds of company websites so you can search them programmatically — by company, industry, or plain English queries.

## What's in a case study?

Every case study is scraped from the source, structured by AI, and stored with these fields:

| Field | Description |
|-------|-------------|
| `title` | The case study headline |
| `summary` | 2-3 sentence overview |
| `featured_client` | The customer company |
| `industry` | The customer's industry |
| `problem` | What challenge they faced |
| `solution` | How it was solved |
| `outcomes` | Results and metrics |
| `tags` | Searchable labels |
| `source_url` | Link to the original |

## Who uses this

- **Sales teams** searching for social proof before a pitch
- **AI agents** doing vendor research or building proposals
- **Analysts** running competitor analysis
- **Procurement teams** evaluating vendors side-by-side
- **Investors** validating a company's customer claims

## How it works

1. We crawl company websites for case study pages
2. AI extracts structured data from each page
3. Embeddings enable semantic search
4. You query via API and get clean JSON back
