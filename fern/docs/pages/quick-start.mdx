# Quick Start

## 1. Get an API key

```bash
curl -X POST https://whatsapp-backend-knd2.onrender.com/v1/keys \
  -H "Content-Type: application/json" \
  -d '{"email": "you@company.com"}'
```

```json
{
  "api_key": "cs_live_abc123...",
  "tier": "free",
  "monthly_limit": 100,
  "message": "Store this key securely — it won't be shown again."
}
```

Save your key. It's only shown once.

## 2. Search case studies

```bash
curl -X POST https://whatsapp-backend-knd2.onrender.com/v1/casestudies/semantic \
  -H "X-API-Key: YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"query": "companies that reduced churn with better onboarding", "limit": 5}'
```

Or use filtered search for exact lookups:

```bash
curl https://whatsapp-backend-knd2.onrender.com/v1/casestudies?company=checkout&industry=ecommerce \
  -H "X-API-Key: YOUR_KEY"
```
