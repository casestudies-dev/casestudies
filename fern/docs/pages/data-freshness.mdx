# Data Freshness

## Coverage

We index case studies from 50+ companies across payments, HR tech, developer tools, cybersecurity, data, ecommerce, cloud, AI, and more.

## Crawl frequency

- New companies added on a rolling basis
- Existing companies re-crawled periodically
- Submitted companies indexed within 48 hours

## How case studies are processed

1. **Crawl** — Firecrawl renders pages including JS-heavy sites
2. **Extract** — Claude AI structures each page into fields
3. **Embed** — Vector embeddings generated for semantic search
4. **Index** — Stored and searchable via the API
